## Running the project

To install dependencies run:
```
npm i
```

For a development server you will need to run:
```
npm start
```

If you are building in dev env run:
```
npm run build
```

To deploy new version to gh-pages run:

```
npm run deploy
```

Check the app at [http://localhost:8080](http://localhost:8080)
